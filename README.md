## 8 вариант
## Задание 3.1.8. Напишите функцию, которая принимает текст и список "стоп-слов". Функция должна заменять все стоп-слова в тексте на символ *.
## Решение:
Для выполнения данной задачи мы обратимся к функции censor_text, которая заменяет запрещенные слова в тексте на звездочки *. Затем нам необходимо разбить слова с помощью split() и очистить каждое слово от знаков препинания: clean_word = word.rstrip('.,!?;:').
Проверяем, есть ли слово в стоп-листе: if clean_word.lower() in [sw.lower() for sw in stop_words], и если слово запрещенное, заменяем на * + сохраняем знаки препинания. В самом конце собираем результат обратно в строку.


Получим:

```
def censor_text(text, stop_words):
    words = text.split()
    result = []
    
    for word in words:
        # Убираем знаки препинания для проверки
        clean_word = word.rstrip('.,!?;:')
        if clean_word.lower() in [sw.lower() for sw in stop_words]:
            # Заменяем слово на одну звездочку
            result.append('*' + word[len(clean_word):])
        else:
            result.append(word)
    
    return ' '.join(result)


text = "Этот плохой пример содержит плохие слова."
stop_list = ["плохой", "плохие"]
print(censor_text(text, stop_list))
```
